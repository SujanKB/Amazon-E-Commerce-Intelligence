# Amazon-E-Commerce-Intelligence

## 📊 Project Overview

This project focuses on analyzing Amazon sales data to uncover valuable insights such as:

* Sales trends over time
* Top-performing products and categories
* Customer purchasing patterns
* Regional sales distribution

The dashboard helps in making data-driven business decisions through clear and interactive visuals.

## 🚀 Features

* 📈 Interactive Power BI dashboard
* 📊 Key KPIs (Total Sales, Profit, Orders, Quantity Sold)
* 🛍️ Category and product-level analysis
* 🌍 Region-wise sales insights
* 📅 Time-series analysis (daily, monthly, yearly trends)
* 🎯 Dynamic filters and slicers

## 🛠️ Tools & Technologies

* **Microsoft Excel**

  * Data cleaning and preprocessing
  * Handling missing and duplicate data
  * Data organization

* **Power BI**

  * Data modeling and relationships
  * DAX calculations
  * Dashboard and report creation

## ⚙️ How to Use

1. Download or clone this repository
2. Open the Excel dataset to understand the raw data
3. Open the `.pbix` file using Power BI Desktop
4. Refresh data if needed
5. Use filters and slicers to explore insights
